Форма авторизации uLogin через социальные сети. Улучшенный аналог loginza.

== Description ==

uLogin — это инструмент, который позволяет пользователям получить единый доступ к различным Интернет-сервисам без необходимости повторной регистрации,
а владельцам сайтов — получить дополнительный приток клиентов из социальных сетей и популярных порталов (Google, Яндекс, Mail.ru, ВКонтакте, Facebook и др.)

== Installation ==

1. Распакуйте архив с плагином в папку `/wp-content/plugins/`
2. Активируйте плагин через меню 'Плагины' а Панели управления.
3. Если панель авторизации не появилась автоматически в форме комментария, пропишите в любом месте темы <?php ulogin_panel(); ?>.

== Frequently Asked Questions ==

= Нужно ли где-то регистрироваться, чтобы плагин заработал? =

Нет, плагин заработает сразу после установки!
